Назначение приложения

Приложение позволяет запоминать слова с помощью импортированной базы данных по средствам сопоставления вариантов ответа.

Условия применения приложения

Приложение может быть использовано если возникает необходимость в запоминании бинарных соответствий. Например, вопросы с единственно правильным ответом, определения, новые слова на иностранном языке и так далее.

Работа с приложением

Для работы с приложением понадобится загрузить файл Excel, содержащий вопросы и ответы для тестирования.

  • В первой колонке таблицы должен содержаться вопрос, в соседней колонке записывается ответ. Минимальное количество вопросов для тестирования - 4.
  • Для запуска тестирования требуется загрузить документ в приложение и выбрать уровень сложности тестирования. От уровня сложности зависит способ ввода ответа.
  • Лёгкий уровень – выбор одного ответа из предложенных вариантов. Сложный – ввод ответа с клавиатуры.
  • Тестирование не имеет логического конца и может быть закончено пользователем в любой момент. После окончания тестирования приложение позволяет скачает статистику ответов, её можно использовать при повторном прохождении тестирования.

Алгоритм работы

  1. На главной странице приложения нажмите кнопку «начать». Приложение перенаправит Вас на страницу тестирования.
  2. Выберите файл Excel с вопросами и ответами, который подготовили заранее (см. Работа с приложением).
  3. Выберите уровень сложности тестировния. В течении тестирования Вы можете менять уровень сложности.
  4. Нажмите кнопку «начать». Появится первый вопрос и поле для выбора/ввода ответа.
  5. Отвечайте до тех пор, пока приложение не спросит нужно ли продолжить тестирование.
  6. Если Ваш ответ «да» - приложение продолжает тестирование. Если «нет» - приложение скачает статистику ответов.
  7. Для повторного прохождения тестирования можно использовать статистику, которая была скачана приложением. Это позволит продолжить обучение, а не начать его заново.
